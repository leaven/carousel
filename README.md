## Carousel
一个基于jquery实现的轮播组件
支持水平滚动、垂直滚动、淡入淡出效果

### JS调用方式
<pre><code>$(".carousel").bdvCarousel({
		containerWidth : 984,
		containerHeight : 500,
		direction : "backward",
		animate : "slide2dVertical"

	});
</code></pre>
